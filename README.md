* keytool
* jarsigner
* Apache2
* Metasploit-Framework

# Installation
```
sudo apt-get install git
git clone https://github.com/yezz123/SpyDroid.git
cd SpyDroid
chmod +x setup.sh
sudo ./setup.sh
chmod +x spydroid.sh
sudo ./spydroid.sh
```
